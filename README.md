# Подготовка к экзамену Telc Deutsch B1

Этот репозиторий содержит материалы для подготовки к экзамену Telc Deutsch B1.

## Структура репозитория

```
telcb1/
├── grammatik/          # Учебники и материалы по грамматике
├── pruefungen/         # Пробные экзамены и тесты
├── themen/             # Заметки и материалы по конкретным темам
├── uebungen/           # Практические упражнения
├── loesungen/          # Ответы и пояснения к упражнениям
├── schreiben/          # Упражнения на письмо
│   ├── uebungen_schreiben.md    # Задания на письмо
│   ├── meine_texte/             # Ваши письменные работы
│   └── korrektur_schema.md      # Схема проверки
└── README.md           # Этот файл
```

## Важные темы для экзамена Telc B1

### 1. Zeitformen (Времена)
- **Präsens** (Настоящее время)
- **Perfekt** (Перфект)
- **Präteritum** (Прошедшее время)
- **Futur I** (Будущее время)

### 2. Nebensätze (Придаточные предложения)
- **dass** (что)
- **weil** (потому что)
- **wenn** (если, когда)
- **obwohl** (хотя)
- **während** (во время, пока)

### 3. Infinitiv mit zu (Инфинитив с zu)
- Инфинитивные конструкции с предлогом "zu"

### 4. Passiv (Пассивный залог)
- Образование и использование пассивного залога

### 5. Konjunktiv II (Конъюнктив II)
- Вежливые формы
- Нереальные условия (irreale Bedingungen)

### 6. Relativsätze (Относительные предложения)
- Придаточные предложения с относительными местоимениями

### 7. Zweiteilige Konnektoren (Двухчастные союзы)
- **nicht nur... sondern auch** (не только... но и)
- Другие двухчастные союзы

### 8. Verben mit Präpositionen (Глаголы с предлогами)
- Глаголы, требующие определенных предлогов

### 9. Adjektivdeklination (Склонение прилагательных)
- Склонение прилагательных в разных падежах

## Материалы

### Грамматика
- `grammatik/Grammatik aktiv A1-B1.pdf` - Учебник по грамматике

### Пробные экзамены
- `pruefungen/Mit Erfolg zu telc Deutsch B1 Testbuch.pdf` - Пробный экзамен

### Упражнения
- `uebungen/` - Папка с практическими упражнениями по каждой теме
  - Отдельные файлы упражнений для каждой из 9 важных тем
  - Смешанные упражнения для комплексной подготовки
  - **Ответы удалены из файлов упражнений** - см. папку `loesungen/`

### Ответы и пояснения
- `loesungen/` - Папка с ответами и пояснениями ко всем упражнениям
  - Отдельные файлы с ответами для каждой темы
  - Подробные пояснения правил грамматики
  - Объяснения типичных ошибок

### Упражнения на письмо
- `schreiben/` - Упражнения на письмо (Schreiben)
  - `uebungen_schreiben.md` - 15 заданий на письмо разных типов
  - `meine_texte/` - Папка для ваших письменных работ
  - `korrektur_schema.md` - Схема проверки текстов
  - **Как использовать**: Напишите текст, отправьте мне для проверки!

## План подготовки

1. Изучите каждую тему из списка выше
2. Выполняйте упражнения из учебника грамматики
3. Решайте пробные экзамены
4. Делайте заметки в папке `themen/` по каждой теме
