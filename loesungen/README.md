# Lösungen (Ответы)

Эта папка содержит ответы и пояснения ко всем упражнениям.

## Структура

Каждый файл соответствует файлу упражнений:
- `01_Zeitformen_loesungen.md` - Ответы к упражнениям на времена
- `02_Nebensaetze_loesungen.md` - Ответы к упражнениям на придаточные предложения
- `03_Infinitiv_mit_zu_loesungen.md` - Ответы к упражнениям на инфинитив с zu
- `04_Passiv_loesungen.md` - Ответы к упражнениям на пассивный залог
- `05_Konjunktiv_II_loesungen.md` - Ответы к упражнениям на конъюнктив II
- `06_Relativsaetze_loesungen.md` - Ответы к упражнениям на относительные предложения
- `07_Zweiteilige_Konnektoren_loesungen.md` - Ответы к упражнениям на двухчастные союзы
- `08_Verben_mit_Praepositionen_loesungen.md` - Ответы к упражнениям на глаголы с предлогами
- `09_Adjektivdeklination_loesungen.md` - Ответы к упражнениям на склонение прилагательных
- `00_Gemischte_loesungen.md` - Ответы к смешанным упражнениям

## Как использовать

1. Выполните упражнения из папки `uebungen/`
2. Проверьте свои ответы в соответствующих файлах этой папки:
   - Для `01_Zeitformen_uebungen.md` → `01_Zeitformen_loesungen.md`
   - Для `00_Gemischte_uebungen.md` → `00_Gemischte_loesungen.md`
   - И так далее для всех тем
3. Изучите пояснения к ошибкам
4. Повторите упражнения, которые вызвали затруднения

## Важно

**Ответы удалены из файлов упражнений!** Все ответы находятся только в этой папке (`loesungen/`).

## Формат ответов

Каждый файл содержит:
- Правильные ответы на все упражнения
- Пояснения к правилам грамматики
- Объяснения типичных ошибок
- Дополнительные примеры
