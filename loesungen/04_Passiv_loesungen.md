# Lösungen: Passiv (Ответы: Пассивный залог)

## Übung 1: Präsens Passiv bilden

**Lösungen:**
1. **wird geöffnet** - Präsens Passiv: werden + Partizip II
2. **wird gelesen** - Präsens Passiv
3. **wird geschrieben** - Präsens Passiv
4. **wird gekocht** - Präsens Passiv
5. **wird repariert** - Präsens Passiv

**Пояснение**: Präsens Passiv образуется: werden (спрягается) + Partizip II. Подлежащее становится объектом действия.

---

## Übung 2: Perfekt Passiv bilden

**Lösungen:**
1. **ist geöffnet worden** - Perfekt Passiv: sein + Partizip II + worden
2. **ist gelesen worden** - Важно: "worden", а не "geworden"!
3. **ist geschrieben worden**
4. **ist gekocht worden**
5. **ist repariert worden**

**Пояснение**: Perfekt Passiv: sein + Partizip II + **worden** (не "geworden"!). Это частая ошибка!

---

## Übung 3: Präteritum Passiv bilden

**Lösungen:**
1. **wurde geöffnet** - Präteritum Passiv: wurde/wurden + Partizip II
2. **wurde gelesen**
3. **wurde geschrieben**
4. **wurde gekocht**
5. **wurde repariert**

**Пояснение**: Präteritum Passiv: wurde (единственное число) / wurden (множественное число) + Partizip II.

---

## Übung 4: Aktiv → Passiv umformen

**Lösungen:**
1. **Die Grammatik wird vom Lehrer erklärt.**
   - Объект "die Grammatik" становится подлежащим
   - Исполнитель "vom Lehrer" с предлогом "von"
   
2. **Die Texte werden von den Schülern gelesen.**
   - Множественное число: "werden"
   
3. **Das Essen wird vom Koch gekocht.**
4. **Das Auto wird vom Mechaniker repariert.**
5. **Der Patient wird von der Ärztin untersucht.**

**Пояснение**: При преобразовании в Passiv объект становится подлежащим, исполнитель указывается с "von" (для лиц) или "durch" (для причин/средств).

---

## Übung 5: von oder durch?

**Lösungen:**
1. **durch** (причина) - Sturm - это причина, не исполнитель
2. **von** (исполнитель) - Postbote - это лицо-исполнитель
3. **durch** (причина) - Krieg - это причина
4. **von** (исполнитель) - Autor - это лицо-исполнитель
5. **durch** (причина) - Regen - это причина

**Пояснение**: 
- **von** - для обозначения исполнителя (лицо): von dem Lehrer, von der Ärztin
- **durch** - для обозначения причины или средства: durch den Sturm, durch den Regen

## Übung 6: Gemischte Zeitformen

**Lösungen:**
1. **wird, geöffnet** (Präsens) - "jetzt" указывает на настоящее время
2. **wurde, gelesen** (Präteritum) - "gestern" указывает на прошедшее
3. **ist, geschrieben worden** (Perfekt) - "bereits" указывает на завершенное действие
4. **wird, gekocht werden** (Futur) - "morgen" указывает на будущее
5. **wird, repariert** (Präsens) - "gerade" указывает на настоящее время

**Пояснение**: Выбор времени зависит от контекста и временных указателей.

## Übung 7: Passiv → Aktiv umformen

**Lösungen:**
1. **Man liest das Buch.** / **Jemand liest das Buch.**
   - Можно использовать "man" (неопределенно-личное) или "jemand"
   
2. **Man öffnete die Tür.** / **Jemand öffnete die Tür.**
3. **Man hat das Essen gekocht.** / **Jemand hat das Essen gekocht.**
4. **Man schreibt den Brief.** / **Jemand schreibt den Brief.**
5. **Man reparierte das Auto.** / **Jemand reparierte das Auto.**

---

## Übung 8: Fehler finden

**Lösungen и пояснения:**
1. ❌ Das Buch wird gelesen worden.
   ✅ **Das Buch wird gelesen.** (Präsens) / **Das Buch ist gelesen worden.** (Perfekt)
   - В Präsens Passiv не используется "worden"

2. ❌ Die Tür ist geöffnet geworden.
   ✅ **Die Tür ist geöffnet worden.**
   - В Perfekt Passiv используется "worden", а не "geworden"!

3. ✅ **Das Essen wird gekocht werden.** (Futur - правильно)
   - Futur Passiv: werden + Partizip II + werden

4. ❌ Der Brief wurde geschrieben worden.
   ✅ **Der Brief wurde geschrieben.** (Präteritum) / **Der Brief ist geschrieben worden.** (Perfekt)
   - В Präteritum не используется "worden"

5. ❌ Das Auto ist repariert geworden.
   ✅ **Das Auto ist repariert worden.**
   - "worden", а не "geworden"!

---

## Übung 9: Übersetzung

**Lösungen:**
1. **Das Buch wird gelesen.** - Präsens Passiv
2. **Die Tür wurde geöffnet.** / **Die Tür ist geöffnet worden.** - Präteritum или Perfekt Passiv
3. **Das Essen wird gekocht.** - Präsens Passiv
4. **Der Brief wurde geschrieben.** / **Der Brief ist geschrieben worden.**
5. **Das Auto wird repariert.** - Präsens Passiv

---

## Übung 10: Sätze bilden

**Lösungen:**
1. **Das Haus wurde gestern gebaut.** - Präteritum Passiv
2. **Die Blumen werden jeden Tag gegossen.** - Präsens Passiv (регулярное действие)
3. **Der Brief ist bereits geschrieben worden.** - Perfekt Passiv
4. **Das Auto wird morgen repariert werden.** - Futur Passiv
5. **Die Tür wird jetzt geöffnet.** - Präsens Passiv
