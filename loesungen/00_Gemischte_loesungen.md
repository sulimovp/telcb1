# Lösungen: Gemischte Übungen (Ответы: Смешанные упражнения)

Этот файл содержит ответы к смешанным упражнениям, которые проверяют знание всех тем.

## Основные правила для смешанных упражнений

1. **Времена**: Präsens, Perfekt, Präteritum, Futur I
2. **Придаточные предложения**: dass, weil, wenn, obwohl, während
3. **Infinitiv mit zu**: после глаголов, прилагательных, существительных (не после модальных!)
4. **Passiv**: werden + Partizip II, в Perfekt: sein + Partizip II + worden
5. **Konjunktiv II**: вежливые формы, нереальные условия
6. **Relativsätze**: относительные местоимения склоняются по падежам
7. **Двухчастные союзы**: nicht nur... sondern auch, sowohl... als auch и т.д.
8. **Глаголы с предлогами**: важно помнить предлог и падеж
9. **Склонение прилагательных**: три типа склонения

**Советы**:
- Внимательно читайте контекст для выбора правильной формы
- Проверяйте порядок слов в придаточных предложениях
- Обращайте внимание на падежи

Полные ответы см. в файлах упражнений. Каждое упражнение проверяет знание нескольких тем одновременно.
