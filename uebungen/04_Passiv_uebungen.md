# Übungen: Passiv (Упражнения: Пассивный залог)

## Übung 1: Präsens Passiv bilden

**Bilde das Passiv im Präsens:**

1. Man öffnet die Tür. → Die Tür ___________ ___________.
2. Man liest das Buch. → Das Buch ___________ ___________.
3. Man schreibt den Brief. → Der Brief ___________ ___________.
4. Man kocht das Essen. → Das Essen ___________ ___________.
5. Man repariert das Auto. → Das Auto ___________ ___________.

## Übung 4: Aktiv → Passiv umformen

**Forme die Sätze ins Passiv um:**

1. Der Lehrer erklärt die Grammatik.
2. Die Schüler lesen die Texte.
3. Der Koch kocht das Essen.
4. Der Mechaniker repariert das Auto.
5. Die Ärztin untersucht den Patienten.

## Übung 5: von oder durch?

**Setze "von" oder "durch" ein:**

1. Das Haus wurde ________ dem Sturm zerstört.
2. Der Brief wurde ________ dem Postboten gebracht.
3. Die Stadt wurde ________ den Krieg zerstört.
4. Das Buch wurde ________ dem Autor geschrieben.
5. Der Unfall wurde ________ den Regen verursacht.


---

## Übung 6: Gemischte Zeitformen

**Setze das Passiv in der richtigen Zeitform ein:**

1. Die Tür ___________ jetzt ___________ (öffnen - Präsens).
2. Das Buch ___________ gestern ___________ (lesen - Präteritum).
3. Der Brief ___________ bereits ___________ (schreiben - Perfekt).
4. Das Essen ___________ morgen ___________ (kochen - Futur).
5. Das Auto ___________ gerade ___________ (reparieren - Präsens).


---
