# Übungen: Konjunktiv II (Упражнения: Конъюнктив II)

## Übung 1: Höflichkeit (Вежливость)

**Bilde höfliche Sätze:**

1. Können Sie mir helfen? → ___________ Sie mir ___________?
2. Können Sie warten? → ___________ Sie bitte ___________?
3. Ich möchte ein Bier. → Ich ___________ ___________ ein Bier.
4. Können Sie das wiederholen? → ___________ Sie das ___________?
5. Können Sie langsamer sprechen? → ___________ Sie ___________ langsamer ___________?


---

## Übung 2: Irreale Bedingungen (Нереальные условия)

**Bilde irreale Bedingungssätze:**

1. Wenn ich reich wäre, ___________ ich ___________ (reisen).
2. Wenn ich Zeit hätte, ___________ ich ___________ (lernen).
3. Wenn es nicht regnete, ___________ wir ___________ (gehen).
4. Wenn du hier wärest, ___________ ich ___________ (glücklich sein).
5. Wenn wir mehr Geld hätten, ___________ wir ___________ (kaufen).
