# Übungen (Упражнения)

Эта папка содержит практические упражнения для подготовки к экзамену Telc B1.

## Структура упражнений

### По темам:
- `01_Zeitformen_uebungen.md` - Упражнения на времена
- `02_Nebensaetze_uebungen.md` - Упражнения на придаточные предложения
- `03_Infinitiv_mit_zu_uebungen.md` - Упражнения на инфинитив с zu
- `04_Passiv_uebungen.md` - Упражнения на пассивный залог
- `05_Konjunktiv_II_uebungen.md` - Упражнения на конъюнктив II
- `06_Relativsaetze_uebungen.md` - Упражнения на относительные предложения
- `07_Zweiteilige_Konnektoren_uebungen.md` - Упражнения на двухчастные союзы
- `08_Verben_mit_Praepositionen_uebungen.md` - Упражнения на глаголы с предлогами
- `09_Adjektivdeklination_uebungen.md` - Упражнения на склонение прилагательных

### Смешанные упражнения:
- `00_Gemischte_uebungen.md` - Смешанные упражнения по всем темам

## Типы упражнений

Каждый файл содержит различные типы упражнений:

1. **Вставка правильной формы** - Setze die richtige Form ein
2. **Образование форм** - Bilde die richtige Form
3. **Преобразование предложений** - Forme die Sätze um
4. **Смешанные упражнения** - Gemischte Übungen
5. **Найди ошибки** - Fehler finden
6. **Перевод** - Übersetzung
7. **Составление предложений** - Sätze bilden
8. **Соединение предложений** - Sätze verbinden

## Как использовать

1. Изучите теорию в папке `themen/`
2. Выполните упражнения по соответствующей теме
3. Проверьте свои ответы в папке `loesungen/` (ответы удалены из файлов упражнений)
4. Изучите пояснения к ошибкам
5. Повторите упражнения, которые вызвали затруднения
6. Решайте смешанные упражнения для закрепления материала

## Советы для подготовки

- Регулярно выполняйте упражнения (хотя бы по 2-3 упражнения в день)
- Делайте заметки о своих ошибках
- Повторяйте сложные темы
- Используйте смешанные упражнения для проверки знаний
- Решайте пробные экзамены из папки `pruefungen/`
