# Passiv (Пассивный залог)

## Образование
- **Формула**: werden + Partizip II
- **Вспомогательный глагол**: werden (спрягается по лицам)

## Präsens Passiv
- **Примеры**:
  - Das Buch wird gelesen. (Книга читается)
  - Die Tür wird geöffnet. (Дверь открывается)

## Perfekt Passiv
- **Формула**: sein + Partizip II + worden
- **Примеры**:
  - Das Buch ist gelesen worden. (Книга была прочитана)
  - Die Tür ist geöffnet worden. (Дверь была открыта)

## Präteritum Passiv
- **Примеры**:
  - Das Buch wurde gelesen. (Книга читалась)
  - Die Tür wurde geöffnet. (Дверь открывалась)

## Использование
- Когда действие важнее, чем исполнитель
- Когда исполнитель неизвестен или неважен
- В официальных текстах, инструкциях

## С предлогом "von" или "durch"
- **von**: для обозначения исполнителя (лицо)
  - Das Buch wird von dem Studenten gelesen.
- **durch**: для обозначения средства/причины
  - Die Stadt wurde durch den Sturm zerstört.
