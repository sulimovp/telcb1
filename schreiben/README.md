# Schreiben (Письмо) - Упражнения

Эта папка содержит упражнения на письмо для подготовки к экзамену Telc B1.

## Как использовать

1. Выберите упражнение из файла `uebungen_schreiben.md`
2. Напишите текст в файле `meine_texte/` (создайте файл с номером упражнения)
3. Отправьте текст мне (AI) для проверки
4. Я проверю:
   - Грамматику
   - Лексику
   - Структуру текста
   - Соответствие заданию
   - Типичные ошибки уровня B1

## Типы заданий на Telc B1

1. **Личное письмо/Email** (80-90 слов)
2. **Формальное письмо** (80-90 слов)
3. **Сообщение/Заметка** (40-50 слов)
4. **Описание ситуации** (80-90 слов)
5. **Выражение мнения** (80-90 слов)

## Критерии оценки

- **Грамматика** (40%): правильность использования грамматических структур B1
- **Лексика** (20%): разнообразие и правильность использования слов
- **Структура** (20%): логичность, связность текста
- **Соответствие заданию** (20%): выполнение всех требований задания

## Советы

- Пишите регулярно (хотя бы 1-2 текста в неделю)
- Используйте разнообразную грамматику (времена, придаточные предложения, пассив и т.д.)
- Обращайте внимание на объем текста
- Проверяйте правописание перед отправкой
