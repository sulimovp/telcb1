# Meine Texte (Мои тексты)

В этой папке создавайте файлы с вашими письменными работами.

## Как использовать

1. Выберите упражнение из `uebungen_schreiben.md`
2. Создайте файл с номером упражнения, например:
   - `uebung_01.md` для первого упражнения
   - `uebung_02.md` для второго упражнения
   - и т.д.

3. Напишите текст в файле

4. Отправьте мне (AI) содержимое файла для проверки

5. Я проверю и дам:
   - Оценку по критериям Telc B1
   - Исправления грамматических ошибок
   - Предложения по улучшению
   - Объяснения ошибок

## Формат файла

Используйте Markdown формат:

```markdown
# Übung 1: Persönlicher Brief

[Ваш текст здесь]
```

## Пример

Создайте файл `uebung_01.md`:

```markdown
# Übung 1: Persönlicher Brief/Email

Liebe Maria,

vielen Dank für deine Einladung zur Geburtstagsfeier! 
Ich freue mich sehr darauf.

Ja, ich kann kommen. Das klingt nach einer schönen Feier.

Soll ich etwas mitbringen? Ich könnte einen Kuchen backen oder 
Getränke kaufen. Sag mir einfach, was du brauchst.

Ich freue mich schon darauf, dich und die anderen Freunde zu sehen!

Viele Grüße,
[Ihr Name]
```
